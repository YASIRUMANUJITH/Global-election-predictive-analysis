# Global-election-predictive-analysis
### A Research-Based Statistical Analysis of Regional Voting Signals and Election Predictability Across Developed and Developing Democracies

## Overview

This project investigates whether the three largest electoral regions (based on 2026 registered elector counts) can reliably predict the final national election winner across Developed and Developing Democracies.

Rather than building a traditional machine learning prediction model, this project follows a research-driven data science methodology by combining:

• Statistical analysis

• Feature engineering

• Exploratory data analysis (EDA)

• Hypothesis testing

• Decision-oriented insights

• Interactive Power BI visualization

The objective is to evaluate whether voting outcomes from the largest electoral regions provide a strong indicator of national election results and to understand how this relationship differs across countries.

## Research Objective

This project answers the following research questions:

• How accurately can the three largest electoral regions predict the national election winner?

• Does prediction accuracy differ between developed and developing countries?

• Does agreement among the largest electoral regions improve prediction reliability?

• How does prediction performance vary across countries?

• What statistical evidence supports these findings?

## Dataset

The dataset was manually compiled and verified from official election sources.

Countries Included

• United States

• Portugal

• Brazil

• South Korea

• Indonesia

• Sri Lanka

### Dataset Characteristics

• 57 presidential election records

• Elections spanning multiple decades

• One row represents one national presidential election

• Largest three electoral regions are fixed according to 2026 registered elector counts for consistency across historical comparisons

## Statistical Methods

The project applies statistical techniques appropriate for a relatively small dataset.

Methods include:

• Descriptive Statistics

• Cross-tabulation

• Accuracy Analysis

• Correlation Analysis

• Mann–Whitney U Test

• Confidence Interval Analysis

• Country-Level Comparative Analysis

## Limitations

The project intentionally fixes the three largest electoral regions using 2026 registered elector counts to ensure a consistent analytical framework across historical elections.

This approach supports cross-election comparison but does not reflect historical changes in electorate sizes over time.

Future work may incorporate year-specific electorate rankings where complete historical data are available.
